# GameUnity
đồng bộ file game unity của team



cách sử dụng git

1: tạo 1 thưc mục để chứa file

2: mở cmd tại thư mục đó

3: gõ vào cmd lệnh:
 - kéo dự án về lần đầu: git clone https://github.com/MrXSchool/GameUnity.git
 - sau khi code xong và muốn báo các thay đổi vào file thì dùng lệnh: git add . ( "." ở đây có nghĩa là thêm tất cả thay đổi. còn nếu chi tiết thì hãy điền file cần add chi tiết vài)
 - ghi chú thích cho các file vừa add bằng lệnh: git commit -m "Nội dung chú thích"
 - và để động bộ file lên dự án của team thì dùng: git push ( nó sẽ đẩy toàn bộ file đã commit lên nhánh chính)

 - git clone: để kéo dự án về lần đầu 
 - git add . : thông báo các chỉnh sử thêm mới
 - git commit : chú thích cho các thêm mới và thay đổi của git add
 - git push : để động bộ file trong máy đã commit lên nhánh chính của team
